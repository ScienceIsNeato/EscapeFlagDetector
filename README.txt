These are the instructions for how to load and use this code. 

COMPILATION:
1. Download and install the most recent version of the Arduino development environment (https://www.arduino.cc/en/Main/Software)
2. Copy the TCS3200D folder into your ../Arduino/Libraries/ folder.
2. If Arduino application is open, restart it.
3. Open EscapeFlagDetector.ino using the Arduino IDE.
4. From the Arduino Menu, go to Sketch->Include Library. You should see the TCS3200D listed as an available library. Select it.
5. Press the Compile button (a check mark). There should be an indication at the bottom of the IDE indicating successful compilation.
